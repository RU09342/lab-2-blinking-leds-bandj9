# Multiple Blink
## Purpose
The purpose of this lab was to make multiple LEDs blink.  This was done exactly how the button blink lab was done, but instead of having one output there were two.  
### Boards Covered
1. MSP430G2553
2. MSP430F5529
3. MSP430FR2311
4. MSP430FR5994
5. MSP430FR6989
